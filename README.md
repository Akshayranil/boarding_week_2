# MoBooM E-Commerce Flutter App

## Project Overview
**MoBooM** is a Flutter-based e-commerce application designed to showcase a **product listing, category filtering, search functionality, and pagination**.  
The app provides a **modern UI with gradient cards, product cards, and real-time interactions**, making it a complete demonstration of Flutter widgets, state management using **Bloc**, and responsive design.

---

## Features

- **Search Products:** Quickly search products using the search bar.  
- **Category Filter:** Filter products by category with a dropdown menu.  
- **Pagination:** Display products with pagination to improve performance and UI clarity.  
- **Gradient Promotional Card:** Highlight promotions using a visually appealing gradient card.  
- **Product Cards:** Each product displays image, title, description, price, and favorite icon.  
- **Favorites Icon:** Add a favorite icon to products (UI only, backend functionality can be added later).  

- **RichText Title:** App title with custom colors and styling.  

---

## Tech Stack

- **Flutter** - UI framework for cross-platform mobile development.  
- **Dart** - Programming language used with Flutter.  
- **Flutter Bloc** - State management for handling product list, categories, and pagination.  
- **Network Image Loading** - Display product images from network URLs.
